# 拉取

## 拉取当前分支的最新更改
git pull

## 拉取指定远程仓库和分支
git pull origin master

# 推

## 查看当前状态和修改
#### 查看工作区状态
git status

#### 查看具体修改内容
git diff

#### 查看暂存区的修改（已add的文件）
git diff --staged

#### 查看每个文件的修改摘要
git diff --stat

#### 查看每个文件的详细修改
git diff --word-diff

## 查看提交历史

#### 查看简洁的提交历史
git log --oneline

#### 查看带分支图的提交历史
git log --oneline --graph --all

#### 查看最近n次提交
git log -n 5

#### 查看某文件的修改历史
git log -p 文件名

## 添加修改到暂存区

#### 添加所有修改（包括新文件）
git add .

#### 添加特定文件
git add 文件名1 文件名2

#### 添加某个目录下的所有修改
git add 目录名/

#### 交互式添加（选择性地添加修改）
git add -p

## 提交修改

#### 提交并添加提交信息
git commit -m "提交描述"

#### 添加所有修改并提交（跳过git add）
git commit -a -m "提交描述"

#### 修改上一次提交（如果忘记添加文件或修改描述）
git commit --amend

## 推入当前分支的更新

#### 推送到默认远端分支（通常是origin master/main）
git push

#### 推送到指定远端和分支
git push origin 分支名

#### 第一次推送并设置上游分支
git push -u origin 分支名

#### 强制推送（谨慎使用！）
git push -f

# 仓库连接配置

## 查看所有远程仓库的简略信息
git remote -v
