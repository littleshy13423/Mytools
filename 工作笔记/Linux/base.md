1. 基础移动
快捷键	功能
Ctrl + A	跳到行首
Ctrl + E	跳到行尾
Ctrl + B 或 ←	向左移动一个字符
Ctrl + F 或 →	向右移动一个字符
Alt + B 或 Ctrl + ←	向左移动一个单词（Backward）
Alt + F 或 Ctrl + →	向右移动一个单词（Forward）
2. 删除操作
快捷键	功能
Ctrl + U	删除光标到行首的内容
Ctrl + K	删除光标到行尾的内容
Ctrl + W	删除前一个单词
Alt + D	删除后一个单词
Ctrl + H (Backspace)	删除前一个字符
Ctrl + D (Delete)	删除后一个字符
3. 历史命令导航
快捷键	功能
Ctrl + P 或 ↑	上一条历史命令
Ctrl + N 或 ↓	下一条历史命令
Ctrl + R	反向搜索历史命令
Ctrl + G	退出搜索模式
4. 其他高效操作
快捷键	功能
Ctrl + L 或 clear	清屏
Ctrl + T	交换当前字符和前一个字符
Alt + T	交换当前单词和前一个单词
Ctrl + _	撤销上一个操作
Ctrl + Y	粘贴最近删除的内容
