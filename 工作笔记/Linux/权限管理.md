## 权限管理

### ACL（Access Control List）可以为特定用户设置访问权限，不改变文件原有权限结构。

给 otheruser 添加 rwx 权限
setfacl -m u:otheruser:rwx /path/to/your/folder

若要递归赋权整个文件夹
setfacl -R -m u:otheruser:rwx /path/to/your/folder

### 修改所有者

```bash
# 将文件所有者改为 user1
sudo chown user1 filename.txt

# 将文件所有者和组都改为 user1
sudo chown user1:user1 filename.txt

# 只修改组（不修改所有者）
sudo chown :group1 filename.txt

# 递归修改目录及其所有内容
sudo chown -R user1:group1 /path/to/directory

# 显示详细操作信息
sudo chown -v user1:group1 filename.txt
```

## 项目 组管理

# 单行命令完成所有操作（替换路径和用户名）
```bash
sudo groupadd gruopname && \
sudo usermod -a -G gruopname liruoyu,user1,user2,user3 && \
sudo chgrp -R gruopname /path/to/your/project && \
sudo chmod -R 775 /path/to/your/project && \
sudo chmod -R g+s /path/to/your/project
```

```bash
sudo chgrp -R gruopname /path/to/your/project
sudo chmod -R g+rwX /path/to/your/project
sudo find /path/to/your/project -type d -exec chmod g+s {} \;
